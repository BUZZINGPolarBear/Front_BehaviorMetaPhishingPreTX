# 🎯 사용 방법 - 실제 뱅킹 앱처럼

## 📱 화면 구성

```
┌─────────────────────────────┐
│ ← 이체              취소     │
├─────────────────────────────┤
│ [위험도 배너]                │  ← 위험도 알림
│  ⚠ 확인이 필요합니다         │
│     상세 ▼                   │
├─────────────────────────────┤
│ 누구에게 보낼까요?           │
│                             │
│ ┌─ 붙여넣은 메시지 ────┐     │  ← 원본 메시지
│ │ 김민수에게              │     │     표시
│ │ 기업 96904420004025   │     │
│ │ 5만원 보내주세요       │     │
│ └───────────────────┘      │
│                             │
│ 🔍 96904420004025          │  ← 계좌번호만
│                             │     파싱됨
│ 최근|내계좌|자주|연락처      │
│                             │
│ 🏢 김민수           ⭐      │
│    기업 9690442...          │
└─────────────────────────────┘
```

## 🚀 테스트 방법

### 1단계: 메시지 복사

다음 메시지 중 하나를 복사하세요:

**저위험 (안전):**
```
김민수님에게
기업은행 96904420004025
5만원 보내주세요
```

**중위험 (주의):**
```
이지은씨
하나 010-28-23037-2
50만원 급하게 송금 부탁드립니다
```

**고위험 (위험):**
```
http://phishing-site.com
계좌: 110-999-888777
200만원 빨리 보내주세요!
급합니다!
```

### 2단계: 계좌번호 입력 필드에 붙여넣기

1. "계좌번호 입력" 필드를 클릭
2. Ctrl+V (또는 우클릭 → 붙여넣기)

### 3단계: 자동 처리 확인

✅ **위에 원본 메시지 표시됨**
- 붙여넣은 전체 메시지가 회색 박스에 표시

✅ **계좌번호만 입력 필드에 파싱됨**
- "96904420004025" 같은 계좌번호만 자동 입력

✅ **위험도 배너 나타남**
- 🟢 녹색: "안전한 거래로 보입니다"
- 🟠 주황: "확인이 필요합니다"
- 🔴 빨강: "주의가 필요합니다"

✅ **금액 자동 추출 (있으면)**
- 메시지에 금액이 있으면 자동으로 저장됨

✅ **연락처 자동 선택 (있으면)**
- 자주 보낸 사람 중 일치하는 사람 자동 선택
- 금액까지 있으면 금액 입력 화면으로 자동 이동

## 📝 상세 시나리오

### 시나리오 1: 안전한 송금

1. 다음 메시지 복사:
```
김민수님
기업 96904420004025
5만원
```

2. 계좌번호 필드에 붙여넣기

3. 결과:
   - 위: "김민수님 기업 96904420004025 5만원" 표시
   - 입력 필드: "96904420004025"
   - 배너: 🟢 "안전한 거래로 보입니다"
   - 자동: 김민수 연락처 선택, 금액 50000원 입력

### 시나리오 2: 주의 필요

1. 다음 메시지 복사:
```
이지은
하나 010-28-23037-2
50만원 급해요
```

2. 계좌번호 필드에 붙여넣기

3. 결과:
   - 배너: 🟠 "확인이 필요합니다"
   - 위험 요인: "고액 거래입니다"
   - 권고: "송금 요청자와 직접 통화로 확인하세요"

### 시나리오 3: 위험 감지

1. 다음 메시지 복사:
```
http://phishing-site.com
계좌 110-999-888777
200만원 긴급 송금
지금 당장 보내주세요!
```

2. 계좌번호 필드에 붙여넣기

3. 결과:
   - 배너: 🔴 "주의가 필요합니다"
   - 위험 요인:
     - "URL이 포함되어 있습니다"
     - "텍스트가 붙여넣기되었습니다"
     - "고액 거래입니다"
   - 권고:
     - "송금 전 반드시 전화로 본인 확인을 하세요"
     - "URL 링크는 클릭하지 마세요"
   - "추가 확인하기" 버튼 표시


## 💡 팁

### 다양한 형식 지원

계좌번호 형식:
- `110-123-456789` ✅
- `1101234567` ✅
- `110 123 456789` ✅

금액 형식:
- `100만원` ✅
- `1,000,000원` ✅
- `금액: 100000` ✅

### 수동 입력도 가능

메시지 없이 직접 계좌번호만 입력해도 됩니다:
- 위험도 분석은 실행 안 됨
- 일반 송금처럼 작동

### 연락처 선택

자주 보낸 사람 리스트에서 직접 클릭해도 됩니다:
- 메시지 붙여넣기 없이도 사용 가능
