# 실제 뱅킹 앱 UI 구현 완료

## 🎉 구현 내용

### 1. 실제 송금 화면 (TransferScreen)

**참고한 실제 뱅킹 앱:**
- 모바일 뱅킹 앱 스타일의 송금 UI
- 자주 보낸 사람 리스트
- 숫자 키패드 입력

**주요 기능:**
- ✅ 연락처 선택 화면
- ✅ 금액 입력 화면
- ✅ 메시지 붙여넣기 자동 파싱
- ✅ 위험도 알림 배너 (송금창 위에 표시)

### 2. 메시지 자동 파싱 (`messageParser.ts`)

**붙여넣기 시 자동 추출:**
- ✅ 금액: "100만원", "1,000,000원" → 자동 입력
- ✅ 계좌번호: "110-123-456789" → 자동 인식
- ✅ 은행명: "달빛", "햇살", "은하" 등
- ✅ 수신인 이름: "홍길동님", "김철수에게"

**예시 메시지:**
```
홍길동님에게
달빛은행 110-123-456789
100만원 보내주세요
```
→ 자동으로 모든 정보 파싱 및 입력

### 3. 위험도 배너 (RiskBanner)

**Calm Security 원칙 준수:**
- ✅ 송금창 위에 자연스럽게 표시
- ✅ 불안 유발하지 않는 표현
- ✅ 위험도별 색상 구분 (Low: 녹색, Medium: 주황, High: 빨강)
- ✅ 펼쳐서 상세 정보 확인 가능
- ✅ 부드러운 애니메이션

**표시 내용:**
- 기본: "안전한 거래로 보입니다" / "확인이 필요합니다" / "주의가 필요합니다"
- 상세: 위험 요인, 권고사항
- 고위험 시: "추가 확인하기" 버튼

### 4. 가명 데이터

**자주 보낸 사람 리스트:**
```typescript
{ name: '김민수', bank: '별빛은행', account: '96904420004025' }
{ name: '이지은', bank: '달빛은행', account: '010-28-23037-2' }
{ name: '박서준', bank: '달빛은행', account: '382-910536-99907' }
{ name: '최유진', bank: '푸른숲은행', account: '35107498278' }
{ name: '정다은', bank: '달빛증권', account: '40093804300' }
```

## 🎨 디자인 특징

### 1. 실제 뱅킹 앱처럼

- **헤더**: 뒤로가기, 타이틀, 취소 버튼
- **검색바**: 계좌번호 입력
- **탭**: 최근 / 내계좌 / 자주 / 연락처
- **리스트**: 아이콘, 이름, 계좌번호, 날짜, 즐겨찾기
- **키패드**: 1~9, 삭제, 0, 완료

### 2. Calm Security

- ❌ "즉시!", "위험!", "심각!" 같은 표현 없음
- ✅ "확인이 필요합니다", "주의가 필요합니다"
- ✅ 부드러운 그라데이션 배경
- ✅ 접을 수 있는 배너 (사용자 선택권)

### 3. 모바일 최적화

- 48px 이상 터치 영역
- 큰 숫자 키패드
- 스와이프 가능한 UI
- 반응형 디자인

## 🚀 사용 방법

### 테스트 시나리오

1. **저위험 메시지 붙여넣기:**
```
김민수님에게
별빛은행 96904420004025
5만원 보내주세요
```
→ 녹색 배너: "안전한 거래로 보입니다"

2. **중위험 메시지 붙여넣기:**
```
이지은에게
달빛은행 010-28-23037-2
50만원 송금 부탁드립니다
```
→ 주황색 배너: "확인이 필요합니다"

3. **고위험 메시지 붙여넣기 (복사본):**
```
[복사] http://phishing.site
계좌: 110-999-888777
200만원 급하게 보내주세요!
```
→ 빨간색 배너: "주의가 필요합니다"
→ "추가 확인하기" 버튼 표시

## 🎯 핵심 차별점

1. **실용성**: 실제 뱅킹 앱처럼 익숙한 UI
2. **자동화**: 메시지 붙여넣기만 하면 모든 정보 자동 입력
3. **침착함**: 불안 유발하지 않고 명확하게 전달
4. **통합성**: 위험도 알림이 송금 흐름에 자연스럽게 통합
