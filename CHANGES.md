## 🎯 현재 상태

### ✅ 작동하는 기능

1. **송금 분석**
   - 텍스트 입력 시 행위 메타데이터 자동 수집
   - 로컬에서 위험도 계산 (0~100점)
   - 위험 수준 판단 (Low/Medium/High)
   - 구체적인 권고사항 제공

2. **추가 검증**
   - 체크리스트 기반 검증
   - 스크린샷 로컬 미리보기
   - 최종 판단 (Allow/Warn/Block)

3. **디자인 시스템**
   - CSS 변수 기반 스타일
   - 일관된 컴포넌트
   - 모바일 최적화

### 🔄 백엔드 불필요

- ✅ API 서버 없이 바로 실행 가능
- ✅ 모든 분석 로직이 프론트엔드에 있음
- ✅ `npm run dev`만으로 즉시 테스트 가능

### 🚀 향후 확장 준비

- Claude API 연동 구조만 준비 (미구현)
- CORS 문제 없이 직접 Claude API 호출 가능
- LLM 기반 고급 분석 추가 가능

## 📝 사용 방법

### 1. 개발 서버 실행

```bash
cd Front_BehaviorMetaPhishingPreTX
npm run dev
```

### 2. 브라우저 확인

- 자동으로 `http://localhost:3000` 열림
- 두 개 탭으로 화면 전환 가능:
  - **송금 분석**: 텍스트 입력 및 위험도 분석
  - **추가 검증**: 체크리스트 및 스크린샷

### 3. 테스트 예시

#### 저위험 (Low)

```
친구에게 5만원 보내주세요
```

#### 중위험 (Medium)

```
110-123-456789로 50만원 보내주세요
```

#### 고위험 (High)

```
[붙여넣기] http://phishing-site.com 계좌 110-123-456789로 200만원 급하게 보내주세요
```
